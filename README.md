# 🔐 Java Web Application – Login & Registration System

A simple **Java web application** that implements **user authentication (Login & Registration)** using **HTML, CSS, and Java Servlets**.

This project demonstrates how frontend forms interact with backend servlets to handle user authentication.

---

## 🚀 Features

* 📝 User Registration (RegisterServlet)
* 🔐 User Login (LoginServlet)
* 🎨 Clean UI with HTML & CSS
* 🔗 Servlet-based backend processing
* 📩 Form handling using POST method
* 📄 Multiple pages (Login, Register, Welcome)

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3
* **Backend:** Java Servlets
* **Server:** Apache Tomcat

---

## 📂 Project Structure

login-system/

│── index.html

│── login.html

│── register.html

│── welcome.html

│── css/

│   └── styles.css

│── LoginServlet.java

│── RegisterServlet.java

│── WEB-INF/

│   └── web.xml

---

## ⚙️ How It Works

1. User registers via `register.html`
2. Data is sent to `RegisterServlet`
3. User logs in via `login.html`
4. Credentials are processed in `LoginServlet`
5. On success → redirected to `welcome.html`

---

## ▶️ How to Run

1. Clone the repository
   git clone https://saddalayogesh.github.io/landing-page/
2. Open in IDE (Eclipse / IntelliJ)

3. Configure Apache Tomcat server

4. Deploy the project

5. Open in browser:
   http://localhost:8080/login-system
